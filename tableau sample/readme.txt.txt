## 📊 Sales Insights Dashboard – Tableau

Built an interactive Tableau dashboard from scratch to visualize regional and category-wise sales performance using CSV data.

🔍 **Features**:
- Bar Chart: Region vs Sales
- Line Chart: Order Date vs Revenue
- Pie Chart: Category by Profit
- Dashboard with filters and interactivity

🛠 Tools: Tableau Public, CSV Data, Custom Filters  
📁 Dataset: `Tableau_SalesData.csv`
